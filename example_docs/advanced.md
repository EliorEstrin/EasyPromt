# Advanced Usage

## Batch Operations

Process multiple items:
```bash
example-cli batch --input file.json
```

## Configuration

Set configuration:
```bash
example-cli config set key=value
```

View configuration:
```bash
example-cli config show
```

## Troubleshooting

Check logs:
```bash
example-cli logs --tail 100
```

Reset configuration:
```bash
example-cli reset --confirm
```
